Web Application Development

Spring 2020

Github

Website

Description

This project was created to minimize the spread of misinformation surrounding the COVID-19 virus. Through real-time database updates, statistical analysis of line charts, and a current affairs news feed, the user can view Coronavirus impacts of the entire U.S. population or any individual state.

Technology Stack

  • - HTML, CSS, JavaScript
  • - Bootstrap
  • - Google Charts & COVID-19 APIs

What I learned

  • - Create and reuse components with Vanilla JS
  • - Async/Await with API calls
  • - Best practices for mobile-first design
