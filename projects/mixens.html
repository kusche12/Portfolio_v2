Description

I created this application as a personal coding challenge. After having completed an online course on mobile development, I decided to put my skills to the test in developing “Mixens” within just 3 weeks. This app allows the user to create, save, and view drinks directly from their phone or tablet.

Technology Stack

  • - React Native
  • - Redux Persist
  • - Google Firebase

What I learned

  • - CRUD operations with Async Storage
  • - Publishing an application to the iOS app store
  • - Rapid Cross-Platform development
