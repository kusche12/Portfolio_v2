Description

This was my final project for INFO 340 Client-Side Development at the University of Washington. By signing into a QuizMe account, the user can create customizable quizzes, save them to their profile, and play them with their friends. Not only did I learn a ton about React development, but now I am a master at Harry Potter trivia!

Technology Stack

  • - React
  • - React Router
  • - Google Firebase

What I learned

  • - CRUD Operations with a backend server
  • - Best practices for user authentication
  • - Building user interfaces with React
  • - Client-Side Routing for Single Page Applications
